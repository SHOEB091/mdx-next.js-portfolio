This is a feature-rich blog application built with the MERN stack. It incorporates Google OAuth for authentication, Firebase for media storage, and both dark and light themes for an enhanced user experience.

## Features

- **User Authentication**: Secure login and signup using Google OAuth.
- **Dark and Light Theme**: Toggle between dark and light themes for a personalized experience.
- **Rich Text Editor**: Create and format blog posts with a user-friendly editor.
- **Media Uploads**: Upload images and videos seamlessly using Firebase storage.
- **Responsive Design**: Fully optimized for all devices.
- **Dynamic Dashboard**: Manage posts, drafts, and user profiles easily.

## Technologies

- **MongoDB**: A NoSQL database for storing users, blogs, and comments.
- **Express.js**: A backend framework for building APIs and server-side functionality.
- **React.js**: A frontend library for building dynamic and interactive user interfaces.
- **Node.js**: A JavaScript runtime for handling server-side operations.
- **Google OAuth**: For secure and efficient authentication.
- **Firebase**: For storing and managing media uploads.
- **Tailwind CSS**: A utility-first CSS framework for responsive and aesthetic design.
