This is a URL shortener project built with Node.js and MongoDB. It provides a user-friendly interface to shorten long URLs and track their analytics.

## Features

- **URL Shortening**: Shorten long URLs to user-friendly links.
- **Click Analytics**: Track the total number of clicks and detailed analytics for each short URL.
- **Authentication**: Secure user accounts with login and signup functionality.
- **Role-Based Access**: Admin and normal users have different permissions.
- **Responsive Design**: Fully responsive interface for all devices.

## Technologies

- **Node.js**: A JavaScript runtime for building scalable server-side applications.
- **Express.js**: A fast, unopinionated web framework for Node.js.
- **MongoDB**: A NoSQL database for storing URLs and user data.
- **EJS**: A templating engine for server-side rendering of dynamic content.
- **JWT**: For secure token-based authentication.
- **Bootstrap**: For responsive and visually appealing UI.
